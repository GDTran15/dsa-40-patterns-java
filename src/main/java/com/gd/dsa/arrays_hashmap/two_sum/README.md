# TWO SUM problem

## Problem analysis
- Input: fixed sized array nums[] and an integer target
- Output: indices of the two numbers add up to target

## Solution
1. Loop through nums array.
2. For each number, calculate the number that we need so that the sum will equal target
3. Check if that number exists in Map if yes return
4. if not put it in

## Big O
- Time complexity: O(n)
- Space complexity: O(n)
